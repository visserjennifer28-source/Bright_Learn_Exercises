# SQL Exercises(1,2 & 3)

# Overview
This repository contains a collection of SQL exercises designed to practice and strengthen core data analysis skills. The exercises focus on querying, filtering, aggregating, and transforming data using SQL.

# Objectives
Improve SQL querying skills
Practice data manipulation and analysis
Understand how to extract meaningful insights from datasets
Topics Covered
SELECT statements

WHERE filtering

JOIN operations

GROUP BY & aggregation functions

ORDER BY & sorting

Subqueries

Tools Used
SQL (MySQL / Snowflake / Hive)

Database tables for practice datasets

# Summary
These exercises demonstrate the ability to work with structured data, perform analysis, and solve real-world business problems using SQL.

This repository reflects my ongoing learning and growth in data analytics.
